BlackTelperion.analyse.unmixing module

Functions for linear unmixing using endmember spectra.

Note that some of these rely on functions implemented in pysptools, which thus may need to be installed for them to work.

BlackTelperion.analyse.unmixing.mix(abundances: BlackData, endmembers: ndarray)[source]

Generate synthetic spectra by linearly mixing an abundance and endmember matrix.

Parameters:

  • abundances – a BlackData instance (e.g. image or cloud) with bands representing abundances (typically from 0 to 1).

  • endmembers – A numpy array of shape (nendmembers, bands), or BlackLibrary instance containing these same endmembers.

Returns:

A BlackData instance with the same type as abundances, but containing the forward modelled (linearly mixed) spectra.

BlackTelperion.analyse.unmixing.unmix(data: BlackData, endmembers: ndarray, method: str = 'nnls')[source]

Perform least squares unmixing to estimate linear combinations of the specified endmembers that best reproduce the observed data.

Parameters:

  • data – a BlackData instance (e.g. image or cloud) to unmix.

  • endmembers – A numpy array of shape (nendmembers, bands), or BlackLibrary instance containing these same endmembers.

  • method – the unmixing constraints. Options are non-negative (‘nnls’, default) or fully constrained (‘fcls’).

Returns:

A BlackData instance with the same type as data, but containing the estimated endmember abundances.

BlackTelperion.analyse.unmixing.endmembers(data: BlackData, n: int, method: str = 'nfindr', **kwds)[source]

Use endmember identification methods implemented in pysptools to find candidate “pure” pixels to use as endmembers. Note that these should always be manually vetted as they represent, in essence, outliers.

Parameters:

  • data – a BlackData instance (e.g. image or cloud) to unmix.

  • n – The number of endmembers to find.

  • method – The endmember identification method. These are fully documented in the pysptools documentation, and can be one of: ‘nfindr’, ‘’ …

Returns:

A BlackLibrary containing the identified endmembers. indices: A numpy array containing the coordinates of the selected endmembers in the input data.

Return type:

library
