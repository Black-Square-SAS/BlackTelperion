BlackTelperion.correct.panel module

NOTE: unsure if this model will be useful, but keep it just in case. Utility class for storing information on calibration panels and combining measured radiance values with known reflectance spectra.

class BlackTelperion.correct.panel.Panel(material, radiance, strict=True, **kwds)[source]

Bases: BlackData

A class for identifying calibration reference in images and storing the observed pixel radiances and known target reflectance values. This is used by, e.g., empirical line calibration procedures.

get_skyview(hori_elev=0.0, up=array([0, 0, 1]))[source]

Get this panels skyview factor. Normal vector must be defined, otherwise an error will be thrown.

Parameters:

  • hori_elev – the angle from the panel to the horizon (perpendicular to the panel’s orientation) in degrees. Used to reduce the sky view factor if the panel is below the horizon (e.g. in an open pit mine). Default is 0.0 (i.e. assume a flat horizon). Can also be negative if sky is visible below the (flat) horizon.

  • up – the vertical (up) vector. Default is [0,0,1].

Returns:

this panels sky view factor (assuming the panel is relatively unoccluded and the horizon is flat).
