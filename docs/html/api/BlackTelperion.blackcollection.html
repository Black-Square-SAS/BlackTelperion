class BlackTelperion.blackcollection.BlackCollection(name, root, header=None, vb=False)[source]

Bases: object

A utility class for mapping data stored in a special file system (.hyc directory or similar) between RAM and disk storage. Useful for (1) reducing IO code and (2) writing out-of-core analyses. The underlying .hyc directory can contain any data that can be read and written by BlackTelperion.io, including: numpy arrays, numbers, strings and BlackTelperion.BlackData instances.
